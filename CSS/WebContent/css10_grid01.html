1. block / inline 요소

block : 좌우 영역을 부모 요소의 전체 크기만큼 잡는다. 상하 영역은 컨텐츠의 크기만큼 잡는다.
inline : 상하 좌우 영역 모두 컨텐츠의 크기만큼 잡는다. inline : 상하 좌우 영역 모두 컨텐츠의 크기만큼 잡는다.

2. block / inline 요소의 height, width

block : width 와 height 를 사용하여 크기를 지정할 수 있다.
inline : width 와 height로 크기를 지정할 수 없다.

3. block / inline 요소의 margin과 padding

block : margin 과 padding이 상하좌우 모두 적용된다.
block : margin 과 padding이 상하좌우 모두 적용된다.
inline : margin 과 padding 이 좌우만 적용된다.
