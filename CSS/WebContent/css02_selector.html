css 기본 선택자

선택자란?
특정한 html 태그를 선택할 떄 사용하는 기능.

1. 태그 선택자

html문서 내에서 같은 태그를 모두 선택할 떄 사용

p{color:blue} 같은 방식으로 사용한다.

2. 아이디 선택자

html 태그의 아이디속성을 사용해서 태그를 선택할 때 사용

#아이디명{color:red;}

3. 클래스 선택자

html 태그의 클래스 속성값을 사용해서 태그를 지정할 떄 사용

.클래스명{color:purple};
