window 객체

		window객체는 자바스크립트의 전역객체로 BOM과 DOM으로 나뉜다.

		BOM : 브라우저 관련된 객체 (location객체, navigator객체, history객체, screen객체)

		DOM : HTML과 관련된 객체 (document객체)

윈도우 객체

		브라우저 창에 대한 설정을 하는 객체

window객체의 timer 메서드

setTimeout()

setInterval()

clearInterval()

screen 객체

운영체제 화면의 속성을 가지는 객체이다.

location 객체

		브라우저 주소 표시줄과 관련된 객체

navigator

웹페이지를 실행하고 있는 브라우저에 대한 정보를 가지고 있는 객체
