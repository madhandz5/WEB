배열

배열이란?

	자바스크립트에서는 자료형 지정이 없기 때문에 모든 자료형을 보관하는 변수의 모음을 배열로 처리한다.

array 객체의 메서드

array 의 메서드들

indexOf()

concat(배열명) : 배열을 결합한다.

join() : 배열을 결합하여 문자열로 반환한다.

reverse() : 배열의 순서를 뒤집는 메서드

sort() : 배열의 정렬

push() : 배열의 맨 뒤에 요소를 추가한다.
pop() : 배열의 맨 뒤의 요소를 제거한다.

shift() : 배열의 맨 앞에 요소 제거.
unshift() : 배열의 맨 앞에 요소 추가

slice() : 배열의 요소를 선택해 잘라내기
splice() 배열의 요소 제거 및 추가
